# ✅ APK успешно собран!

## 📱 Как скачать APK

### Шаг 1: Откройте GitHub Actions
**Ссылка:** https://github.com/suninvest777/mallware/actions

### Шаг 2: Найдите завершенную сборку
- Найдите workflow run с зеленой галочкой ✅
- Нажмите на него

### Шаг 3: Скачайте APK
- Прокрутите вниз до раздела **"Artifacts"**
- Нажмите на **"app-apk"**
- Скачайте ZIP архив
- Распакуйте и получите APK файл

**Имя файла:** `photoview-1.0-*.apk` (например, `photoview-1.0-arm64-v8a-debug.apk`)

---

## 📲 Установка на Android

### Вариант 1: Прямая установка
1. Передайте APK файл на Android устройство (через USB, email, облако)
2. Откройте файл на устройстве
3. Разрешите установку из неизвестных источников (если нужно)
4. Установите приложение

### Вариант 2: Через ADB (для тестирования)
```bash
adb install photoview-1.0-*.apk
```

## ⚙️ Настройка

### Изменить изображение:
1. Замените `ready_app/embedded_image.jpg` на свое изображение
2. Замените `ready_app/photo.jpg` на то же изображение
3. Запустите сборку заново (push в GitHub)

### Изменить Telegram ID:
1. Откройте `ready_app/app.py`
2. Найдите строку: `TELEGRAM_USER_ID = 8350084460`
3. Измените на свой ID
4. Запустите сборку заново

### Изменить токен бота:
1. Откройте `ready_app/config.json`
2. Измените `"bot_token": "ВАШ_ТОКЕН"`

## 🔄 Пересборка APK

Если нужно изменить что-то:

1. **Измените файлы** в `ready_app/`
2. **Сделайте commit и push:**
   ```bash
   git add ready_app/
   git commit -m "Изменения в приложении"
   git push
   ```
3. **GitHub Actions автоматически соберет новый APK**
4. **Скачайте новый APK** из Artifacts
