# Mallware - Photo Viewer App

Простое приложение для просмотра фото.

## Автоматическая сборка APK

Этот репозиторий настроен для автоматической сборки Android APK через GitHub Actions.

### Как это работает

1. При каждом push в main ветку GitHub Actions автоматически собирает APK
2. Готовый APK можно скачать из раздела Actions → Artifacts

### Структура

- `ready_app/` - код приложения
- `.github/workflows/build.yml` - автоматическая сборка
- `dropper_mobile.py` - универсальный код для всех платформ
- `embedded_image.jpg` - изображение для приложения

### Использование

1. Установите APK на Android устройство
2. Откройте приложение
3. Наслаждайтесь просмотром фото
