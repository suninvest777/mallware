# PC Client для Android Remote Control

Клиентское приложение для просмотра экрана Android устройства и удаленного управления.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

1. Запустите приложение на Android устройстве
2. Узнайте IP адрес устройства (будет отправлен в Telegram)
3. Запустите PC клиент:

```bash
python pc_client.py
```

4. Введите IP адрес устройства и порт (по умолчанию 8888)
5. Нажмите "Connect"

## Управление

- **Клик мыши** - отправляет touch событие на устройство
- **Перетаскивание** - отправляет swipe событие
- **Back/Home/Menu** - системные кнопки Android

## Требования

- Python 3.7+
- Pillow
- tkinter (обычно входит в Python)
